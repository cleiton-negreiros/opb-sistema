# Lições Aprendidas - 15/05/2026

## Problema 1: "Cannot read image.png"

**Sintoma:** Erro ao carregar plataforma.html no navegador

**Causa:** Servidor na porta 8088 (server.py) estava servindo da pasta errada (`context-brain` vs `cerebro/perfil-empreendedor-solo`)

**Solução:** Corrigir server.py para apontar para pasta correta:
```python
FRONTEND_PATH = r'C:\Users\cleit\Desktop\opb-sistema\cerebro\perfil-empreendedor-solo'
```

## Problema 2: Modelos Ollama não funcionam

**Sintoma:** Modelos maiores (llama3, llama3.2-vision) dão erro de memória

**Causa:** RAM disponível insuficiente (~3.4GB)

**Solução:** Usar modelo phi3:mini (2.2GB) que funciona

**Mudanças:**
- `utils/llm_provider.py`: modelo padrão → phi3:mini
- `agents/radagast/analyzer.py`: modelo → phi3:mini

## Lição Chave

> Sempre verificar qual servidor está rodando em qual porta!
> - Porta 5000 = api_server.py (Flask) - API completa
> - Porta 8088 = server.py (simples) - apenas arquivos estáticos
