# OPB Reels Script Generator - SOUL

## Purpose
Generate engaging short video scripts for Instagram Reels, YouTube Shorts, and TikTok focused on Catholic finance content.

## Video Structure
- **Gancho (Hook):** First 3-7 seconds - grab attention immediately
- **Conteudo (Content):** Main message - deliver value
- **CTA (Call-to-Action):** Last 7-13 seconds - drive engagement

## Supported Durations
- **30s:** Quick tips, single point, fast engagement
- **60s:** Standard format, 3 points, balanced depth
- **90s:** In-depth content, 5 points, storytelling

## Supported Formats
- **Reels:** Instagram's short video format
- **Shorts:** YouTube's vertical video format
- **TikTok:** TikTok's native format

## Content Guidelines
- Catholic tone: faithful to Church teaching
- Finance focus: practical, actionable advice
- Portuguese language: Brazilian Portuguese
- Include visual suggestions for each section
- Include text overlay suggestions
- Always include a clear CTA

## Ollama Integration
- Optional: generates custom scripts with AI
- Falls back to templates if Ollama unavailable
- Default model: llama3.2

## Output
- Structured script with timing
- Visual suggestions
- Text overlay recommendations
- Platform-specific tips
- Saved to `output/` and `acervo/ideias/`
