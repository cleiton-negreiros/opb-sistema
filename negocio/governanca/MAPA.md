# Governança — Mapa

> Camada de governança do cérebro. É aqui que mora a autoridade — o que pode e o que não pode, como decidimos, o que aprendemos.

## Subpastas

| Pasta | Para quê |
|-------|----------|
| `regras/` | Regras-mãe consolidadas de comportamento (1 arquivo por tema) |
| `decisoes/` | Decisões estratégicas datadas (`AAAA-MM.md`) |
| `licoes/` | Lições aprendidas (incidentes + acertos) |
| `referencias/` | Fichas de APIs, sistemas externos, contas |
| `contexto/` | Missão, produtos, métricas-norte, stack, roadmap |
| `sessoes/` | Sessões arquivadas |
| `auditorias/` | Relatórios de auditoria |
| `evidencias/` | Provas (prints, exports) |

## Quando abrir

- Antes de propor mudança de processo: leia `regras/`.
- Antes de tomar decisão grande: leia `decisoes/` recentes.
- Toda sexta: revise `licoes/` do mês.
