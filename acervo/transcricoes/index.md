# Transcricoes de Videos

> Transcrições de vídeos do YouTube

## Como usar

```bash
python main.py "URL_DO_VIDEO"
python main.py --listar
python main.py --ler "nome-do-video"
python main.py --buscar "palavra-chave"
```
